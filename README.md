# homebrew-mcp-linker

Homebrew tap for [MCP Linker](https://github.com/milisp/mcp-linker) - One-click add and sync MCP servers across AI clients.

## What is MCP Linker?

**MCP Linker** provides instant setup for Model Context Protocol (MCP) servers. Click → Add → Done. No manual config editing or LLM calls required.

### Key Features

- **Local Sync** — Sync MCP server config across Multi-Client
- **600+ Curated MCP Servers** — Built-in marketplace
- **Multi-Client Support** — Claude Desktop/Code, Cursor, VS Code, Cline, Windsurf, Codex, Roo Code
- **Cross-Platform** — macOS, Windows, Linux
- **Smart Detection** — Auto-detect Python, Node.js, uv environments

## Installation

```bash
brew tap milisp/mcp-linker
brew install --cask mcp-linker
```
